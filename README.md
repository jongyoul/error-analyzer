# How to build

## Local
```bash
# Copy .envrc.template to .envrc and edit the file
cp .envrc.template .envrc

# Source .envrc. You can skip it if you use `direnv`
. .envrc

./gradlew runApp
```

## (WIP) Publish
```bash
# Copy .envrc.template to .envrc and edit the file
cp .envrc.template .envrc

# Source .envrc. You can skip it if you use `direnv`
. .envrc

./gradlew publishApp
```
